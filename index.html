MODELING AND PREDICTION OF CONCRETE COMPRESSIVE STRENGTH USING MACHINE LEARNING

International Journal of Engineering Research and Development - Impact Factor: 8.154

  • Employed machine learning techniques such as Multilayer Perceptron Neural Network (MLP) and Support Vector Machines (SVM) to predict concrete compressive strength.
  • Achieved approximately 90.07% accuracy using MLP and 90.22% accuracy using SVM, demonstrating the effectiveness of these models.
  • Utilized k-fold cross-validation with k set to 5 to evaluate the generalization performance and validate the robustness of machine learning models, achieving an average accuracy of 88.5% across all folds.
